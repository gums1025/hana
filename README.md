# Explainable AI Portfolio — Landing Page

Market validation landing page for an **Explainable AI Investment Portfolio Explanation Service**.

## Local

Open `index.html` directly in a browser.

## Deploy (GitHub → Vercel)

1. Create a GitHub repository (empty).
2. Push this folder to GitHub.
3. In Vercel: **Add New → Project → Import Git Repository**, then select the repo.
4. Framework preset: **Other** (or static). Build command: **None**. Output: **/ (root)**.

Vercel will deploy on every push to the connected branch.
